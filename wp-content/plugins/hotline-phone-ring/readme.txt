== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Hotline Phone Ring'
3. Activate Hotline Phone Ring from your Plugins page.

=== Manually ===

1. Upload the `hotline-phone-ring` folder to the `/wp-content/plugins/` directory
2. Activate the Hotline Phone Ring plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
